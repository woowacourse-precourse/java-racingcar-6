## 자동차 경주 기능 목록

- [ ] 경주 할 자동차 이름 입력 기능
  - [x] 쉼표(,)를 기준으로 구분하는 기능
    - [ ] 유효성 검증 기능
      - [ ] **예외처리**
        - [x] 빈값을 입력한 경우
        - [x] 이름의 길이가 6자 이상인 경우
        - [x] 중복된 자동차 이름을 입력하는 경우
- [ ] 이동 횟수 입력 기능
  - [ ] 유효성 검증 기능
    - [ ] **예외처리**
      - [ ] 빈값을 입력한 경우
      - [ ] 숫자가 아닌 경우
      - [ ] int 형 값의 범위를 넘어선 경우
- [ ] 0에서 9 사이에서 무작위 값을 구하는 기능
- [ ] 각차수 별 실행 결과 출력 기능
- [ ] 우승자 출력 기능
  - [ ] 한명 &rightarrow; 자동차 이름 출력
  - [ ] 여러명 &rightarrow; 쉼표(,)를 이용하여 구분

### 예외처리 공통 조건
- 사용자가 잘못된 값을 입력할 경우
  - `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료.
