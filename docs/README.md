## 🔨 기능 요구 사항
### 입력
**자동차 이름 입력**
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능
- 공백을 입력할 경우 `IllegalArgumentException` 발생
- 구분을 쉼표로 하지 않은 경우 `IllegalArgumentException` 발생
- 자동차 이름이 5자 이상인 경우 `IllegalArgumentException` 발생

**이동횟수 입력**
- 사용자는 몇 번의 이동을 할 것인지를 입력
- 숫자가 아닌경우나 공백인 경우 `IllegalArgumentException` 발생
- 0을 입력할 경우 `IllegalArgumentException` 발생

### 게임 진행
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈추기 가능
- 전진하는 조건은 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우
- 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력

### 최종 결과
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려줌(우승자는 한 명 이상)
- 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분
