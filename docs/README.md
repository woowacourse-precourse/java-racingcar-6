# 2주차 미션 - 자동차 경주

## 자동차
- 전진 또는 멈춤
- 이름(5자 이하)
- 전진할 때 이름을 같이 출력
- 0에서 9까지 중 랜덤 숫자를 구하고 이 숫자가 4 이상일 경우 전진
- 

## 사용자
- 자동차에 이름 부여
- 몇번의 이동을 할 것인지
- 사용자가 잘못된 값을 입력한 경우 `IllegalArgumentException` 


## 구현 시 주의사항
- indent는 2까지만 허용(그 이상은 메소드로 처리)
- 3항 연산자는 사용하지 말기
- 메소드가 한가지 일만 처리할 수 있도록 함
