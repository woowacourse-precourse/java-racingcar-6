# 🏎️ 자동차 경주 게임 🏎️

## 📉 클래스 다이어그램

## 📄 기능 요구 사항
    경주 할 자동차 이름 입력받음
    시도할 횟수 입력받음
    전진 시도 횟수만큼 메소드 구현
    전진 조건은 0~9사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우
    각 차수별 진행 결과 출력
    게임 완료하면 누가 우승했는지 알려줌, 우승자는 한 명 이상 가능
    사용자 입력이 잘못됐을 경우 예외터뜨림
