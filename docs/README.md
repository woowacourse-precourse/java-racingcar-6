🚀 구현 기능 목록

#### 입력

1. 경주 할 자동차 이름(이름은 쉼표(,) 기준으로 구분)
2. 몇번의 이동을 할 것인지 시도한 횟수(자연수)
3. (예외처리) 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료

#### 출력

1. 실행 결과 출력(-로 전진 결과 출력)
2. 최종 우승자 출력
    - 공동 우승자 출력 가능

#### 자동차

1. 자동차 이름 생성
    - (예외처리) 이름 5자 이하
    - (예외처리) 이름이 중복된 경우
    - (예외처리) 이름을 입력 하지 않은 경우
2. 자동차 전진 기능
    - 0-9 사이에서 무작위 값을 구한다.
    - 무작위 값이 4 이상일 경우 전진 가능
3. 자동차 이동 기능
    - 전진시 +1
