## 구현 기능

### 1. 자동차 이름 및 시도 횟수 입력

- [ ] 경주할 자동차 이름 입력
  - [ ] ',' 를 기준으로 구분
  - [ ] 5자 이하로 제한
- [ ] 시도할 횟수 입력
  - [ ] 0 이상의 정수만 입력 가능

### 2. 경주 실행 및 반복
- [ ] 각 자동차 별로 0 ~ 9 사이의 랜덤 값을 생성
  - [ ] 4 이상일 경우만 전진
- [ ] 시도 횟수만큼 반복

### 3. 우승자 출력
- [ ] 가장 멀리 간 자동차 출력
  - [ ] 여러 대일 경우 쉼표(,)로 구분
