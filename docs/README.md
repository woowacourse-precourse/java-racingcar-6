# 자동차 경주 게임

## ☀ 구현할 기능 목록

### 사용자 입력
- 경주할 자동차 이름을 입력 받는다.(쉼표 구분 형식)
- 시도할 횟수를 입력 받는다.
  - 예외) 숫자가 아닌 경우
- 사용자가 잘못된 값을 입력하는 경우 `IllegalArgumentException`을 발생시킨 후 애플리케이션을 종료한다.

### 게임 실행 전 초기화
- 입력 값을 검증한다.
  - 예외) 자동차의 각 이름이 5자 초과인 경우
- 사용자 입력을 기반으로 자동차 객체 정보 업데이트한다.

### 게임 실행
- 각 차수마다 랜덤 값을 초기화 한다.
  - 각 자동차마다 랜덤 값을 생성한다.
    - 조건) 전진 : n >= 4
    - 조건) 정지 : n < 4
- 랜덤 값을 바탕으로 각 차수마다 실행 결과를 계산한다.
- 각 차수마다 실행 결과를 출력한다.

### 결과 출력
- 게임 결과에 따라 최종 우승자 출력한다.
  - 단독 우승의 경우
  - 중복 우승의 경우
