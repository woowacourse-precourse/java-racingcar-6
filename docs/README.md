# 🚗 미션 - 자동차 경주

## ✏️ 기능 목록
- n대의 자동차 이름 입력
    - [x] 쉼표(,)를 기준으로 구분
    - 잘못된 입력: `IllegalArgumentException` 발생 후 종료
      - [ ] 빈 문자열인 경우
      - [ ] 5자 이하가 아닌 경우
- 몇 번의 이동할 것인지 입력
  - [x] 게임을 진행할 횟수 입력
  - 잘못된 입력: `IllegalArgumentException` 발생 후 종료
    - [ ] 빈 문자열인 경우
    - [ ] 양의 정수가 아닌 경우
- 각 차수별 실행 결과 출력
  - [x] 각 자동차마다 0~9 사이의 랜덤 값 생성
  - [x] 랜덤 값이 4 이상인 경우 해당 자동차 전진
  - [x] 자동차 이름과 함께 결과 출력
- 최종 우승자 출력
  - [x] 가장 많이 전진한 우승자 출력
