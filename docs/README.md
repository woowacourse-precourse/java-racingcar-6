# 미션 2 - 자동차 경주

### 🚀 기능 목록
#### - 자동차 경주 후 최종 우승자 출력

    1. 경주 할 자동차 이름 입력 (쉼표 기준으로 구분, 5자 이하)
    
    2. 몇 번 이동할 지 입력
   
     3. 자동차는 전진 혹은 멈춤 
        - 전진 조건 : 무작위 값 생성 (4이상 시 전진)
    
    4. 경주 완료 후 우승 자동차 출력 (여러 명일 시 쉼표로 구분)
    
    5. 잘못 입력 시, IllegalArgumentException 발생 후 종료
        - 자동차 이름 입력 길이 제한 (1 ~ 5자)
        - 자동차 이름 중복
        - 자동차 이름 빈칸
        - 이동 횟수 최소 제한
