# 🚘 미션 - 자동차 경주

## ✅ 구현할 기능 목록
### 1. 사용자의 입력
- 각 자동차의 이름을 쉼표(`,`)를 기준으로 입력
  - 자동차의 이름은 5글자 이하만 가능
  - 입력되는 자동차는 총 n대
- 이동 횟수 입력

### 2. 자동차 전진
- 모든 자동차에 대해 입력된 이동 횟수 만큼 전진 여부 확인
  - 0 ~ 9 사이의 무작위 수를 구해 4 이상일 경우 전진⭕ ️
  - 4 미만일 경우 전진 ❌
- 입력된 이동 횟수 만큼 진행 되었 다면 종료
- 출력 형태는 아래와 같음
  ```
  {자동차1}: --
  ```
### 3. 우승자 판별
- 입력된 이동 횟수 만큼 진행 된 후 우승자 판별
- 전진 횟수를 비교 하여 최다 전진 횟수인 경우 우승
- 1대인 경우 자동차 이름 출력
- 2대 이상인 경우 `,`로 구분하여 출력
