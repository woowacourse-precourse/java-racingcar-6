# 필요한 기능 목록

Car
- [ ] : 자동차의 이름을 입력받아 자동차를 생성한다.
  - [ ] : 자동차의 이름이 5자 이하인지 확인한다.
- [ ] : 자동차는 정해진 규칙에 따라 이동한다.
- [ ] : 자동차는 이름과 이동거리를 올바르게 출력한다.

Cars
- [ ] : 자동차 목록을 초기화한다.
- [ ] : 자동차 목록을 출력한다.
- [ ] : 우승자 목록을 계산한다.
- [ ] : 게임을 한 차례 진행한다.

Game
- [ ] : 반복 횟수만큼 라운드를 진행한다.
- [ ] : 게임에 필요한 정보를 초기화한다.

OutputView
- [ ] : 우승자 목록을 출력한다.

InputView
- [ ] : 자동차의 이름을 입력받는다.
- [ ] : 시도할 횟수를 입력받는다.
