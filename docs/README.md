### 구현할 기능 목록

---
**메소드가 한 가지 일만 하도록 만들기**
- 사용자로부터
  - 자동차 이름 입력받는 메소드
  - 시도할 횟수를 입력받는 메소드
  - 입력받은 문자열이 조건에 맞는지 확인하고 아닐시 예외를 반환하는 메소드

- 출력
  - 각 차수별 실행 결과를 출력하는 메소드
  - 자동차의 전진 결과를 출력하는 메소드
  - 우승자(단독은 이름만, 공동 우승자는 쉼표로 구분)를 출력하는 메소드


- 자동차 경주
  - 자동차가 조건에 따라 전진, 정지하는 메소드
  - 레이스를 실행하는 메소드
  - 가장 멀리 전진한 차량의 이름을 반환하는 메소드
