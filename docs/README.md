# 기능 구현 목록

## Race
- [x] 주어진 횟수만큼 레이스 진행한다.

## Registration
- [x] 이름을 입력 받아 자동차 생성한다.
- [x] 사용자에게 전진시도 횟수를 입력받는다.

## InputValidator
- [x] 이름이 5자 이하인지 체크한다.
- [x] 정수인지 체크한다.
- [x] 0보다 큰 수인지 체크한다.

## Car
- [x] 무작위 값을 구해 전진한다
- [x] 시도할 때 마다 결과를 출력한다

## Judgment
- [x] 전진거리를 비교한다.

## Referee
- [x] 우승자를 선정한다.
