## 🚀 프로젝트 설명
초간단 자동차 경주 게임을 구현한다.

## ✏️ 기능 정리

### 입출력 요구사항

#### 입력
* 경주할 자동차 이름 ( 이름은 ,로 구분 )
* 예외처리 : 이름은 5자 이하, 1대 이상
> pobi, woni, jun
* 시도할 횟수
> 5   

#### 출력
* 각 차수별 실행 결과 (전진, 멈춤)
* 전진하는 조건 : 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우
> pobi : --   
> woni : ----   
> jun : ---
* 우승자 안내 문구
> * 단독   
> 최종 우승자 : pobi
> * 공동 ( ,로 구분 )   
> 최종 우승자 : pobi, jun

### 실행 결과 예시
> 경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)   
> pobi,woni,jun   
> 시도할 회수는 몇회인가요?   
> 5
