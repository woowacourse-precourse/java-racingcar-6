최대한 기능별로 나누도록
1. 자동차 생성 (이름 부여) - Car 
2. 전진 관련 조건 (몇번 이동하는가? 최대의 거리) - Move에 같이? 일단 하면서 수정
3. 자동차 이동 (random값 추출) - Move
4. 이동할때마다 출력 하기 - Track
5. 이동거리에 따른 결과값 출력 - Finish
6. 결과값을 통해 우승자를 출력 - Winner
7. 예외 처리


사용자 입력값 - Input
