# 기능 목록

## 진행 흐름

- 게임이 시작되는 메시지를 출력한다. (입력 메시지)
- 경주할 자동차의 이름을 부여한다.
- 자동차가 몇 번 이동할 지 입력한다.
- 0 ~ 9까지 랜덤한 숫자 중 4 이상일 경우 자동차가 전진한다.
- 게임이 진행된 상황을 보여준다.
- 이를 사용자가 입력한 횟수만큼 반복한다.
- 사용자가 입력한 회수만큼 진행 됐다면 최종 우승자를 발표한다.
- 게임이 종료된다.

- [x] 전진하는 기준

    - 변수명 : MINIMUM_NUMBER_FOR_FORWARD
- [x] 랜덤 수의 MINIMUM 값

    - 변수명 : RANDOM_MAXIMUM_RANGE

- [x] 사용자 입력 메시지

## 예외사항

- 자동차 이름은 5자 이하만 가능하다.
- 시도할 회수 입력에 숫자가 아닌 다른 수가 들어오면 안 된다.
