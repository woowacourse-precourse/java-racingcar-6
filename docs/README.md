기능 설계
1. n개 자동차 입력 받기
2. 입력값 판별하기
3. 몇번 시행할지 입력
4. 랜덤 수 생성
5. 랜덤 수 범위 판별
6. 유효한 값이라면 이동칸 +1
7. 이동칸에 따라 출력하기
8. 모든 시행이 끝났다면 객체간 비교
9. 1등 출력하기
