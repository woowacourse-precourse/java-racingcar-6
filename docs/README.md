구현할 기능 목록
----------------

- [x] Player들이 자동차들의 이름을 쉼표를 기준으로 구분하여 입력하는 기능

- [x] 이동할 회수를 입력하는 기능

- [x] 입력한 이름과 회수를 저장하고 오류를 검증하는 기능
    - [x] 이름은 5자 이하만 가능
    - [x] 회수는 숫자만 가능

- [ ] 컴퓨터가 랜덤한 숫자를 생성하는 기능

- [ ] 입력한 회수만큼 랜덤한 숫자를 생성할 때 마다 0에서 9사이에서 무작위 값이 4 이상일 경우 전진하는 기능

- [ ] 누가 우승했는지 알려주는 기능 여러 명일 경우 쉽표를 이용하여 구분
