# 우아한 테크코스 프리코스 2주차
## 🏎️자동차 경주
### 입력 요구사항
#### 자동차 이름
- 자동차 이름은 쉼표(,)를 기준으로 구분
- 자동차의 이름은 5자 이하만 가능
    - [x] 6자 이상의 이름 예외처리
    - [x] 1글자 미만 이름 예외처리
    - [x] 중복된 이름 예외처리

#### 자동차 이동 횟수
- 숫자만 입력
    - [x] 숫자가 아닌 입력 예외처리
___  
### 기능 요구사항
- [x] 자동차 이름 입력 문구 출력
- [x] 자동차 이름 입력 받기
- [x] 시도할 횟수 입력 문구 출력
- [x] 시도할 횟수 입력 받기
- [x] 0 ~ 9 사이의 랜덤 값 추출
- [x] 추출된 랜덤 값이 4 이상일 경우에만 전진
- [x] 각 차수별 실행 결과 출력
- [x] 게임이 완료되면 우승자 출력
- [x] 우승자는 1명 이상도 가능
- [x] 우승자가 여러명인 경우 쉼표(,)를 이용해서 구분
- [x] 잘못된 값이 입력된 경우 IllegalArgumentException 발생 후 프로그램 종료
___
