# 자동차 경주

## 📍 구현 기능 목록

***

**1. 게임 시작 문구 출력**

**2. 컴퓨터 : 0~9 사이의 무작위 값 선택**

**3. 사용자 : 경주에 참여할 자동차 이름 입력**

- 예외
  - 각각 자동차 이름이 5자를 초과하는 경우
  - 중복된 이름이 입력되는 경우
  - 아무것도 입력하지 않은 경우
  - 자동차 이름이 쉼표(,)로 구분되지 않은 경우

**4. 사용자 : 1~9 사이의 희망하는 경주 횟수 입력**

- 예외
    - 숫자가 아닌 것(ex. 문자열)을 입력한 경우
    - 아무것도 입력하지 않은 경우
    - 범위를 벗어난 값을 입력한 경우

**5. 실행 결과 출력**

**6. 최종 우승자 출력**
