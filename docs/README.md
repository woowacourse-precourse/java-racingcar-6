자동차 경주게임
n대의 자동차.
클래스는 Class 2개.
한개는 Car 클래스, 다른 한개는, Car들을 입력받고,
입력받은 Car로 레이싱 경주를 하도록 만드는 Gamestart클래스.

Gamestart클래스 안에는 setCar, setRound, printResult 사용
setCar로 입력받음과 동시에 Car세팅.
setRound로 입력받음과 동시에 Round세팅.
printResult로 결과를 프린팅.
