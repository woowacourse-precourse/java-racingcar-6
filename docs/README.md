## 기능 목록

- [X] : 사용자 입력 처리
  - [X] : 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료
  - [X] : 몇 번 이동 할 것인지 입력받기
    - [X] : 양수만 가능
  - [X] : 경주 할 자동차 입력받기
    - [X] : 쉼표(,)를 기준으로 구분
    - [X] : 5자 이하만 가능

- [ ] : 게임 진행 
  - [ ] : 0-9 사이에 무작위 값 중 4이상일 경우 전진
  - [ ] : 모든 자동차의 이동이 끝나면 각 차수별 실행결과 출력

  ``` 
  car1 : -
  .
  .
  .
  carN : -
  ```

- [ ] : 우승자 가려내기
  - [ ] : 여러 명일 경우 쉼표(,)로 구분

## 객체 목록
- [ ] 자동차
  - [ ] : 전진하기
  - [X] : 이름
  - [X] : 현재 위치
- [ ] 게임
  - [X] : 현재 상태
  - [X] : 자동차 리스트
  - [X] : 이동 횟수
  - [ ] : 현재 상태에 따른 행위
    - [X] : 준비 - 사용자 입력 처리
    - [ ] : 진행 - 게임 진행
    - [ ] : 종료 - 우승자 가려내기
