# 구현할 기능 목록

### 입력

- [x] 사용자 이름 입력
  - [x] 예외 - 범위 밖 길이의 사용자 이름이 입력되는 경우 (0 이하, 6자 이상)
- [x] 시도할 회수 입력
  - [x] 예외 - 숫자가 아닌 문자가 입력되는 경우
  - [x] 예외 - 범위 밖의 숫자가 입력되는 경우 (0 이하, 1000 이상)

### 출력

- [x] 실행 결과 출력
  - [x] 입력된 순서대로 출력
- [x] 최종 우승자 출력
  - [x] 우승자가 중복인 경우에, 입력된 순서대로 출력

### 경주 게임

- [x] 각 사용자 이름의 차 생성
- [x] 전진할 조건인 랜덤 변수 생성
- [x] 조건에 따라 차 전진
