# 기능 구현 사항

* 시스템 출력
  * 경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
  * 시도할 회수는 몇회인가요?
  * 실행 결과
  * 최종 우승자 :

* 사용자 입력
  * 자동차 이름을 쉼표 기준으로 구분
    * 5글자를 넘어가면 `IllegalArgumentException` 발생 후 종료
  * 시도할 횟수
    * 숫자가 아닌 값이 입력되면 `IllegalArgumentException` 발생 후 종료

* 게임 로직
  * 0 ~ 9 사이의 숫자 중 하나를 랜덤으로 생성
  * 각 자동차마다 랜덤으로 생성한 수 중 4이상이 나오면 한칸 전진
  * 가장 많이 전진한 자동차가 우승
