## 자동차 경주

---

### 기능 목록

- ✅ 사용자 입력
    - ✅ 자동차 이름 입력(쉼표(,) 기준으로 구분)
    - ✅ 시도할 회수 입력
    - ✅ 사용자 입력 예외 처리
        - ✅ 자동차 이름 입력 시 5자 이하가 아닌 경우
        - ✅ 자동차 이름 입력 시 공백을 입력한 경우
        - ✅ 시도할 회수 입력 시 숫자가 아닐 경우
- ✅ 전진 조건에 따른 자동차 전진 기능
- ✅ 결과 출력
    - ✅ 각 차수별 실행 결과 출력
    - ✅ 최종 우승자 안내 문구 출력(공동 우승 가능)
