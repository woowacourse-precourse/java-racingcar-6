## 기능 구현 목록
### 1. 자동차 이름 입력 받기
  - 1-1. 이름은 쉼표 기준으로 구분한다.
  - 1-2. _[예외]: 이름이 5글자를 넘은 경우_
  - 1-3. 시도할 라운드 횟수 입력받기
  - 1-4. _[예외]: 정수가 아닌 다른 입력을 넣은 경우_
### 2. 자동차 이동/전진 게임 
  - 2-1. 무작위 값을 구해 4 이상인 경우 전진
  - 2-2. 한 라운드 당 모든 자동차가 1회씩 동작
  - 2-3. 한 라운드 당 결과를 출력
### 3. 게임 종료
  - 3-1. 모든 라운드가 끝나면 게임 종료
  - 3-2. 최종 우숭자를 출력한다.
    - 우승자는 여러 명일 수 있다.
