**2주차 미션 - 자동차 경주**

---

### 기능 구현 목록

1. **사용자**의 자동차 이름/시도 횟수(몇번 이동할지) 입력
- 5자 이하
- 쉼표(,) 기준으로 구분 
<br>


- 예외처리) 사용자가 잘못된 값 입력할 경우 IllegalArgumentException 후 애플리케이션 종료
  - 자동차 이름 5자 초과일 경우 
  - 자동차 이름이 1개 이하일 경우
  - 시도 횟수 : 자연수가 아닐 경우 (0, 음수, 문자 등)

2. 전진 또는 멈춤
- 전진 : 0-9 사이 무작위 값 구한 후 4 이상일 경우.
- 그 외 경우(3이하)는 멈춤. 

3. 경주 실행 결과 출력
- 3의 결과와 1의 시도횟수를 이용

4. 우승자 출력
- 한 명 이상 가능
- 여러명일 경우 쉼표(,)이용해 구분
