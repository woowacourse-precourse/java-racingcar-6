# 기능 목록
- n대의 자동차 이름 입력(이름 사이 쉼표(,)를 기준으로 구분, 5자 이하)
    - 잘못된 값을 입력하는 경우 IllegalArgumentException을 발생하도록 구현
- 시도할 횟수 입력(자연수 입력)
    - 잘못된 값을 입력하는 경우 IllegalArgumentException을 발생하도록 구현
- 전진 조건에 따라 차수별 실행 결과 출력에 반영
    - 전진 하는 조건 구현
      - 자동차별로 0~9사이의 무작위값을 구해 4이상인 경우 전진
      - 4미만인 경우 멈춤
    - 차수별 실행 결과 출력
      - 자동차 : 전진횟수 * "-"로 출력되도록
- 각 차수별 딜레이시간 삽입
- 경주가 끝난 후 최대로 전진한 자동차 출력
  - 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분
