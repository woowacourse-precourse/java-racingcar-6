# 우테코 프리코스 2주차 미션 - 자동차 경주

## 🔍 기능 구현


### 🚀 random_race()

* input_car()에서 입력된 값을 바탕으로 레이싱을 수행한다.
* 각 자동차 별로, 랜덤 숫자를 도출하여 4 이상인 경우에 전진할 수 있다. 
* 전진한 값은 저장되며, 각 시행마다 하이픈(-) 의 반복으로 출력된다.

### 🚀 input_car()

* 차의 이름과 시행 횟수를 입력받는 함수이다. 
* 각 자동차에 이름을 부여할 수 있다. 
* 자동차 이름은 5글자 이하이며, 쉼표(,)로 구분된다.
* 이름 입력 후, 몇번 경기를 수행할 지 숫자도 입력한다.
* 자동차 이름이 5글자를 초과하거나 / 아무것도 입력하지 않거나 / 횟수가 숫자가 아닌 경우를 판별하여 IllegalArgumentException()를 throw한다.
* 입력된 자동차 이름과 시행 횟수는 Map에 저장된다. 

### 🚀 winner_is()

* random_race()가 끝나면, Map에 저장된 자동차의 전진 횟수를 카운트하여 우승한 자동차의 이름을 발표한다.

### 🛵 main 함수

1. input_car() 함수로 자동차의 이름과 시행 횟수를 입력받는다. 
2. random_race() 함수로 자동차의 경주를 시행하고 저장한다. 
3. winner_is() 함수로 자동차 경주의 결과를 안내한다.
