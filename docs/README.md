* 게임시작
* 자동차 이름 받기
  * 쉼표(,)를 기준으로 구분함.
  * 이름은 5자 이하
  * 값에 공백이 포함되어 있는지
* 이동횟수 값 받기
  * 숫자 인지
  * 값이 0 인지
* 이동로직
  * 0에서 9사이 랜덤 값
  * 4이상이면 전진
  * 이동 시 자동차 이름과 - 출력
* 결과 출력
  * 첫 번째로 도착하면 우승
  * 둘 이상일 경우 쉼표(,)로 구분
* 추가 요구 사항 
  * 들여쓰기 depth가 3이 넘지 않았는지, 2까지 허용 
  * 3항 연산자 쓰지 않음
