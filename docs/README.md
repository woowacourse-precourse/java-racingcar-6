# 구현 기능 목록

---
- 사용자 입력 받는 기능
  - 검증 기능
    - 자동차 이름 입력
      - 이름은 쉼표 기준으로 구분되며 한번에 입력받는다
      - 이름은 5자 이하 여야한다
    - 시도 횟수 입력
      - 숫자만 입력 가능하다

- 출력 하는 기능
  - 실행 결과 출력
    - 자동차의 이동을 '-'로 시각적으로 보여준다
  - 최종 우승자 출력
    - 우승자가 두 명일 경우 쉼표로 구분하여 출력

- 자동차 전진 가능 판단 기능
  - return: true / false

- 전진할 경우 모델에 반영 기능

- 최종 우승자 선별 기능
