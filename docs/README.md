### 구현할 기능 목록 정리

1. 사용자 이름 입력 검증
   - 쉼표(,)를 기준으로 각 이름이 5자 이내인지 확인. 그렇지 않으면 예외 발생
   - 만약 중복되는 이름이 있다면 예외 발생 처리 (명세에 없으므로 자율 구현)

2. 사용자의 이동 횟수 입력 검증
   - 양수를 제외한 입력은 예외 발생 처리 
ㅈ
3. 매 차수마다 전진하는 로직 수행
   - 무작위로 뽑은 수가 4이상이면 앞으로 이동 
   - 매 차수마다 전진 기록 출력

4. 모든 차수가 끝나면 우숭자 결정
   - 가장 많이 전진한 우승자 선택
   - 만약 여러 명이라면 공동 우승자 처리 
   - 우승차 목록 출력
