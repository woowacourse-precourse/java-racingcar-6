+ #### 2주차 개인적 목표

    + 저번 주차에서 영어로 쓰다 이도저도 아니게됨.
  + **객체지향 프로그래밍** 연습하기

  + 손보다 머리를 많이 쓰기
---
+ #### 공통 피드백 정리
  + 1주차 공통 피드백은 크게 `git` 관련 피드백, 컨벤션 관련 피드백이 주를 이루고 있다.
  + `git` 관련 피드백
    + git으로 관리할 파일들에대한 기준
    + 의미있는 커밋메시지로 작업 내용을 직관적으로 이해 할 수 있게한다.
    + PR 보내기전 브랜치확인
  + 컨벤션관련 피드백
    + 이름 축약금지 
    + 공백 남발금지 ----> 문맥을 분리하는 부분에 사용
    + 의미 없는 주석 금지 ----> 주석 대신 네이밍을 통해 최대한 의도를 드러낼것.
---
+ #### 그외 2주차 신경쓸 사항

  + 3항 연산자 금지.
