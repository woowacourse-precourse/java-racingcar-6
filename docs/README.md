## 요구 사항
* 자동차 입력
* 시도 횟수 입력
* 자동차 이름은 쉼표(,)를 기준으로 구분하며, 5자 이하만 가능.
* 전진하는 조건은 0~9 사이의 무작위 값을 구한 뒤, 4이상일 경우만 전진.
* 시도 횟수가 종료되면 누가 우승했는지 출력
* 우승자는 여러명이 될 수 있고, 여러명일 경우 쉼표(,)로 구분

## Exception
* 자동차의 이름이 문자가 아닌 경우
* 자동차의 이름이 중복되는 경우
* 자동차의 이름에 공백이 들어가는 경우
* 자동차의 이름이 없는 경우
* 자동차의 이름이 5자 넘어가는 경우
* 시도 횟수가 0이상의 정수가 아닌 경우
