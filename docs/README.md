- [x] 자동차가 랜덤하게 전진 혹은 멈춤 
  -[x] 0~9 무작위의 값 생성 : generateRandomNumber()
  -[x] 0, 1, 2, 3 : 멈춤 / 4, 5, 6, 7, 8, 9 : 전진 : isCarMoving()
- [x] 쉼표 기준 자동차 이름 구분 : splitCarNames(String input)
- [x] 자동차 이름은 5자 이하 : validateCarNameLength(String carName)

- [] 우승자 목록 생성

  - [] 여러명 : 쉼표 구분 출력
