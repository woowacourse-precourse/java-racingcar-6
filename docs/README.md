# 기능 구현 리스트

1. 게임을 실행한다
   시작 텍스트 출력
2. 자동차는 이름을 가진다.
   - 글자 수 길이 설정 : 5자 이하
   - 구분 : 이름은 쉼표(,) 기준으로 구분
   - 중복을 허용하지 않는다

3. 각 자동차는 경주한다.
   - 입력 : 몇 번 이동할 지 입력
   - 이동 : 난수 값이 4이상이면 전진

4. 경주
   - 0~9 난수 생성
   - 멀리 이동한 자동차가 승리
   - 공동 우승 (쉼표( , ) 구분)
     - [ex] ‘최종 우승자 : pobi, jun’
   - 이동 횟수 제한 추가
5. 최종 우승자 출력
