
초간단 자동차 경주 기능 구현 요구 사항 정의

# 자동차
- [ ] 이름, 거리를 입력할 수 있어야 한다.
- [ ] 거리를 수정할 수 있어야 한다.

# 게임
- [ ] 여러 대의 자동차의 이름을 입력할 수 있어야 한다.
  - [ ] 이름이 5글자를 초과한 경우 IllegalArgumentException 을 발생시키고 게임을 종료한다.
- [ ] 게임에서 시도할 회수를 정할 수 있어야 한다.
- [ ] 각 차수별 결과를 출력한다.
- [ ] 우승자를 출력한다.
  - [ ] 우승자를 선별할 수 있다.
  - [ ] 우승자가 여러명일 경우 모두 출력할 수 있어야한다.
