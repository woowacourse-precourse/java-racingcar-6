# 구현 기능 목록

----

### 경주 관련 기능
1. 사용자 출력 화면
   1. 경주 관련 사용자 인터페이스 
      1. 참가자 명단 입력 인터페이스
      2. 경주 횟수 입력 인터페이스
   2. 결과 출력 기능
      1. 진행 결과 출력 인터페이스
      2. 최종 우승자 출력 인터페이스
2. 내부 동작 기능
   1. 입력 횟수에 따른 경주 진행 기능
   2. 우승자 산출 기능
3. 예외 처리
   1. 사용자가 잘못된 값을 입력했을 경우
      1. 경주 횟수
### 드라이버 및 자동차 관련 기능
1. 자동차 관련 기능
   1. 입력받은 값에 따른 전진, 후진 기능
2. 드라이버 관련 기능
   1. 운전 기능
      1. 랜덤한 값을 추출해 전진, 후진 결정
3. 예외 처리
   1. 사용자가 잘못된 값을 입력했을 경우
      1. 참가자 이름
