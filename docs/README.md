##우테코 프리코스2주차 - racingcar

  1. Input Class
     사용자에게 입력받을 값에 대해 설명을 출력하고 값을 입력받는 클래스
  2. ValidateInputName Class
     입력받은 값에 대해 검증하는 클래스
     
  3. Car Class
     Car인스턴스를 만드는 클래스
  4. ScoreBoard Class
     실행 결과를 출력하기 위한 클래스
  5. Move Class
     자동차가 조건을 만족할 때 1칸씩 전진하는 기능을 구현한 클래스
  6. Winner Class
     우승자를 추출하고 출력하는 클래스
