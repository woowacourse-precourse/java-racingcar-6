# 숫자 야구 게임
## 기능목록

1. 자동차 이름 입력 기능
   - 입력 받은 이름 유효성 검사
      - 5자리 이하인 이름
      - 영어로 이루어진 이름
2. 입력 받은 이름 분리 기능
   - (,)쉼표를 기준으로 분리
3. 시도 횟수 입력 기능
   - 입력 받은 값 유효성 검사
      - 숫자 입력값
4. 랜덤 값 생성 기능
   - 0 ~ 9 중 한자리 랜덤한 값
5. 생성된 랜덤값으로 전진 유무 파악 기능
   - 4이상일 경우, 한 칸 전진하기
6. 한 횟수가 끝났을 때, 자동차 별 전진 결과 출력 기능
7. 주어진 시도 횟수만큼 4 ~ 6 반복 기능
8. 최종 결과 출력 기능
