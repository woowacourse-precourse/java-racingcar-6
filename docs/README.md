### 기능 목록

1. 자동차 객체 생성
   - 자동차 이름 설정 기능 (5자 이하)
   - 자동차의 초기 위치 설정

2. 입력 처리
   - 자동차 이름 입력 및 유효성 검사
   - 시도 횟수 입력 및 유효성 검사

3. 경주 로직 구현
   - 무작위 값 생성 (0-9)
   - 전진 조건 판단 (무작위 값이 4 이상일 때 전진)
   - 각 라운드별 자동차 위치 업데이트

4. 경주 결과 출력
   - 각 라운드별 자동차 위치 출력
   - 최종 우승자(들) 출력

5. 예외 처리
   - 잘못된 입력에 대한 예외 처리 및 메시지 출력
