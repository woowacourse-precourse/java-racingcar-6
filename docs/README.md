# 구현 목록
- 자동차 이름 입력 안내 메시지를 출력한다.
- 시도 회수 입력 안내 메시지를 출력한다.
처리 결과 안내 메시지를 출력한다.
- 자동차 이름 입력 기능
  - 공백 입력 불가
  - 5자 이하만 가능
- 경주 회수 입력 기능
  - 양수만 가능(문자열 X)
- 전진 여부 판단
  - 0 ~ 9 사이에서 무작위 값을 뽑아서 4 이상인 경우 1을, 미만인 경우 0을 반환
- 우승 자동차 판정 기능
  - 시도할 횟수를 모두 끝낸 후, 가장 많이 전진한 자동차가 우승한다.
  - 우승자가 2명이상인 경우, 콤마를 넣어 출력
