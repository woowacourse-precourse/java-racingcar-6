# 기능 목록 정리

## 자동차 입력을 받습니다.
- 문구: 경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
- 잘못된 입력시 IllegalArgumentException 발생
    - 이름은 1글자 이상 5글자 이하 가능
    - 이름의 시작은 공백 불가

## 시도 횟수를 입력 받습니다.
- 문구: 시도할 회수는 몇회인가요?
- 잘못된 입력시 IllegalArgumentException 발생
  - 자연수만 입력 가능

## 각 시도마다 각 자동차에 대한 전진 판단, 전진, 전진 결과 출력
- 각 자동차 마다 0 ~ 9 사이의 무작위 정수를 뽑음
- 4이상의 숫자가 나온 경우 자동차는 1 칸 전진
- 전진 결과 출력

## 시도 횟수를 끝낸 후 승리자 출력
- 단독 우승자
  - "최종 우승자 : pobi"
- 최종 우승자
  - "최종 우승자 : pobi, jun"
