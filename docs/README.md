# 기능 요구 사항 정리

1. 경주 할 자동차 이름 입력 받기
    2. 쉼표로 구분해서 입력을 받음
3. 차수 입력 받기 (전진 시도 할 횟수)
4. 실행결과를 출력 => 차수별로 출력을 한다.
5. 차수가 끝나면 최종 우승자 출력
    6. 우승자가 여러명 일 시 ,로 구분

# 전진 조건

1. 0 ~ 9까지 랜덤 숫자를 굴린다.
    2. 4 이상이면
        3. 전진
    4. 4 미만이면
        5. stop

# 입력 예외 사항

1. IllegalArgumentException 발생 후 Application 종료

# 클래스 분류

1. Car 클래스
    2. 모델 역할 수행
    3. 자동차의 상태(이름, 위치) 및 동작(전진) 정보를 가지고 있음
2. CarView 클래스
    3. 뷰 역할 수행
    4. 출력을 담당(자동차 상태, 최종 우승자)
3. RacingGameController 클래스
    4. 컨트롤러 역할 수행
    5. 사용자 입력 및 게임 로직 실행
    6. createCar 메서드 -> 자동차 이름 입력받고 Car 객체 생성
