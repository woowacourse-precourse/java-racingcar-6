# 기능 구현 목록
- [X]   사용자 입력 : 경주할 자동차 이름, 시도할 횟수
  - 이름은 쉼표(,)로 구분
  - 잘못된 입력 시 IllegalArgumentException 발생시킨 후 종료
  - 예외 처리
    - case 1. 각 자동차 이름은 5자 이하
- [X] 자동차 별로 전진하는 조건을 위한 무작위 값 구하기
  - 무작위 값이 4 이상일 경우 1칸 전진, 아니면 멈춤
- [ ] 각 차수별 실행 결과 출력
  - -로 전진 표현
  ```
    pobi : - 
    woni :
    jun : -
  ```
- [ ] 최종 우승자 구한 후 출력
  - 우승자가 여러 명일 경우 쉼표(,) 이용해 구분
