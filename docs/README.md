# MVC 패턴을 사용한 자동차 경주

## Domain
### 자동차 도메인 
    [⭕] 이름, 전진 거리 변수
---
## Repository
### 자동차 리스트 Memory DB
    [⭕] 자동차 객체 저장

    [⭕] 이름으로 자동차 객체 찾기
    
    [⭕] 모든 자동차 내역 출력
---
## Controller
### 사용자 입력
    [⭕] 이름 입력
    
    [⭕] 회수 입력

### 동작
    [⭕] 회수만큼 전진 거리 갱신 루프
---
## Service
### 자동차 저장
    [⭕] 이름 문자열 자르기
    
    [⭕] Repository에 저장하기

### 자동차 - 전진 여부 정하기
    [⭕] 랜덤 값으로 전진 여부 판단
    
    [⭕] 전진 거리 갱신

### 우승자 이름 전달
    [⭕] 우승자 판단
    
    [⭕] 우승자 전달
---
## View
### 출력
    [⭕] 입력 안내 문구 출력
    
    [⭕] 차수별 진행 결과 출력
    
    [⭕] 우승자 결과 출력
---
