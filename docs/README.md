# 자동차
각 자동차에 이름을 부여하는 메서드(,)로 구분 generateCarName() ( O )
-> ,로 이름을 구분지어서 받는 로직 ( O )

몇회 이동할지 받는 메서드 moveTry() ( O )
랜덤한 숫자 Randoms.pickNumberInRange(0,9); 에서 4 이상시 - 를 추가하는 메서드 moveOne() ( O )
자동차 n대중 -갯수가 가장 많은 자동차의 이름이 승자로 출력 (공동우승 가능) printWinner()
