# woowacourse-racingcar

## 구현할 기능 목록

- main
   - Game
     - 게임 전체 플로우(이름 입력, 시도 횟수 입력, 실행결과 출력) 메소드 구현
       - 자동차 이름 옳지 않은 입력 예외 처리
       - 사용자 이동 횟수 옳지 않은 입력 예외 처리
   - Generate
     - 자동차 이름 추출 메소드 구현
       - 자동차 이름 5자 초과 예외 처리
       - 중복된 자동차 이름 예외 처리
     - 각 차수 경주 결과 출력 메소드 구현
     - 차수 경주 결과 바(-) 표현 메소드 구현
     - 경기 결과 출력 메소드 구현
   - Race
     - 각 차수 경주 진행 메소드 구현
     - 전진 조건 판정 메소드 구현

- test
    - GameTest
      - 자동차 이름 옳지 않은 입력 예외 처리 테스트
      - 시도할 횟수 옳지 않은 입력 예외 처리 테스트
    - GenerateTest
      - 중복 이름 있는 경우 예외 처리 테스트 
      - 바(-) 개수 테스트
      - 최종 우승자 0명 테스트
      - 최종 우승자 1명 테스트
      - 최종 우승자 2명이상 테스트
    - RaceTest
      - pickNumberInRange_0에서_9까지의_정수_중_한개의_정수 반환 테스트
    - HashMapTest
      - HashMap_put_테스트
      - HashMap_반복문_테스트
      - HashMap_replace_테스트
