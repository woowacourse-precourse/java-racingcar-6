# 미션 - 자동차 경주

## 구현할 기능 목록
- validateName
    - 이름 예외 처리
- car
    - car의 이름 설정
    - car의 이름 반환
    - randomNumber가 4이상이면 전진(position++) / move
    - position반환
    - round마다 출력 메소드
- race
    - 사용자가 입력한 수만큼 round진행
    - 해당 게임의 car 모두 반환
    - 랜덤한 수 뽑아서 car.move로 전달
- getWinners
    - 최대 position의 car들의 이름 반환
