# 🏎 미션 - 자동차 경주

---

## 🗒 기능 구현 목록

### ✔ 자동차 생성 기능
- 사용자가 입력한 값으로 자동차를 생성한다.
  - [**makeCar**] 메시지 ➡ 사용자 입력 문자열 (List<String>)

### ✔ 자동차 이동 기능
- 랜덤 숫자에 따라 자동차가 멈출지 이동할지 결정한다.
  - [**moveCar**] 메시지 ➡ 랜덤 숫자 (int)

### ✔ 우승자 선정 기능
- 자동차 경주 게임을 완료하고 누가 우승했는지를 알려준다.
  - [**winner**]

### ✔ 시도 횟수 입력 기능
- 사용자가 시도 횟수를 입력한다.
  - [insertTryTimes]

--- 
## [유효성 검사 기능]
### ✔ 자동차 이름 유효성 검사 기능
- 사용자가 입력한 자동차 이름이 1자 이상, 5자 이하인지 검사한다
  - [**validateCarName**] 메시지 ➡ 자동차 이름 (String)

--- 

## [입출력 기능]
### ✔ 자동차 이름 입력 기능
- 쉼표(,)를 기준으로 구분하여 자동차 이름을 입력한다.

### ✔ 이동 결과 출력 기능
- 시도 횟수마다 이동 결과를 출력한다.
  - [**printResult**] 메시지 ➡ 자동차 이름 (List<String>), 자동차 위치 (List<Integer>)

### ✔ 최종 우승자 출력 기능
- 최종 우승자를 출력한다
  - [**printWinner**] 메시지 ➡ 자동차 이름 List<String>
