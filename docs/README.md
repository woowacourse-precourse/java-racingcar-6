> ⁜ **우아한테크코스 2024 프리코스 2주차 미션**
>
> 🌼 _Implemented by_ `yummygyudon` (**정동규**)

## 🛠 Feature List
### 1️⃣ 게임 처리 기능
- [x] 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- [x] 각 자동차에 이름을 부여할 수 있다. 
  - [x] 자동차 이름은 쉼표(,)를 기준으로 구분
  - [x] 이름은 5자 이하만 가능
- [x] 사용자는 **몇 번의 이동을 할 것인지를 입력**할 수 있어야 한다.
- [x] **전진** : **4 이상**일 경우 (0 ~ 9 사이 **무작위 값**)
  - [x] `camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()`를 활용한다
  - [x] 0에서 9까지 랜덤한 수를 선택한다. 
- [x] 자동차 경주 게임을 **완료한 후 누가 우승했는지**를 알려준다. 
  - [x] 우승자는 **한 명 이상**일 수 있다.
  - [x] 우승자가 여러 명일 경우 **쉼표(,)를 이용하여 구분**한다.
  - 단독 우승자 안내 문구
    ```
    최종 우승자 : pobi
    ```
  - 공동 우승자 안내 문구
    ```
    최종 우승자 : pobi, jun
    ```
- [x] 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다.
- [x] 게임 플레이어가 입력한 횟수만큼 진행된 결과를 출력한다.
- [x] 최종 우승자를 출력한 후, 게임이 종료된다.
- [x] 게임을 종료한 후 게임을 다시 시작하는 기능은 없으며 완전히 종료된다.

<br/>

> 자동차 입력 예시
```java
/**
 * 입력 : pobi,woni,jun
 */

// 아래와 같이 모든 차의 이름과 함께 결과에 출력되어야 한다.
pobi : -
woni :
jun : -
```
> 이동 횟수 입력 예시
```java
/**
* 입력 : 2
  */

// 아래와 같이 모든 차의 이름과 함께 결과에 출력되어야 한다.
pobi : -
woni :
jun : -

pobi : --
woni : -
jun : -

```

<br/>

### 2️⃣ 입/출력 기능
- [x] 경주할 자동차의 이름을 입력받는다.
  - Message : `경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)`
- [x] 시도할 횟수를 입력받는다.
  - Message : `시도할 회수는 몇회인가요?`
- [x] 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
  - Message : `실행 결과` 
  - 최초 1회만 출력된 후, 회차별 경주 결과만 출력한다. 
- [x] 입력한 횟수만큼 진행된 결과를 출력한 후, 마지막에 최종 우승자를 출력을 끝으로 게임이 종료된다.
  - Message : `최종 우승자 : ` 
