# ✏️ ️ 구현기능

<br>

- [X] 동작
  - [X] 자동차의 이름을 입력 받는다.
  - [X] 횟수를 입력받는다.
  - [X] 자동차는 전진, 멈춤을 통해 횟수만큼 진행한다.
  - [X] 전진하는 조건은 4이상 부터다.
  - [X] 각 차수별로 게임을 동작한다.

<br>

- [X] 예외처리
  -[X] 자동차는 쉼표를 기준으로 구분한다.
  - [X] 5자 이하만 가능하다.
  - [X] 글자 외에 공백이 들어와서는 안된다.
  - [X] 시도횟수에는 숫자말고는 들어와서는 안된다.
  - [X] 시도 횟수 숫자에는 음수를 넣지 못한다.

<br>

- [X] 출력
  - [X] 우승자는 1명 이상이 가능하다.
  - [X] 우승자의 거리와 함께 출력을 한다.
