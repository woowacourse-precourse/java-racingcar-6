# 기능 목록

---

## 입력 요구사항
- [ ] 사용자 입력
    - [ ] 자동차 이름
        - `n` 개의 이름 입력
        - 각 이름은 `,`로 구분
        - 자동차 이름은 5자 이하

    - [ ] 시도할 회수 입력

 ---
    
## 기능 요구사항
- [ ] 자동차 별 `0 ~ 9` 사이 무작위 값 추출
    - 전진 조건: 값이 4 이상일 경우
- [ ] 경주가 종료된 후, 우승자(들) 구하기
    - 경주 종료 시점에서 가장 멀리 간 자동차(들)

---
    
## 출력 요구사항
- [ ] 안내문구
- [ ] 실행결과
    - `{name} : -` 와 같이, 하이픈으로 전진 횟수를 출력.
- [ ] 최종 우승자

---

## 예외처리
- [ ] 자동차 이름
    - 아무런 값을 입력하지 않을 시
    - 입력받은 이름이 5글자를 초과할 시
- [ ] 시도할 회수
    - 제약 조건은 없지만, `0 ~ Integer.MAX_VALUE`로 제한
    - 여유가 된다면, `BigInteger` 사용
