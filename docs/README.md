# <구현 기능 목록>

### 1. 사용자 입력 기능

- [x] 참가하는 자동차들 입력 받기
- [x] 진행할 라운드 수 입력 받기

### 2. 입력 검증(예외처리) 기능

- [x] 자동차 이름 입력길이 검증
- [x] 자동차 이름 중복 검증 (추가)
- [x] 자동차 이름 통합 검증 (추가)
- [ ] ~~라운드 수 입력 검증(삭제)~~

### 3. 출력 기능

- [x] 한 라운드 결과 출력
- [x] 최종 우승자 출력

### 4. 자동차 이동 기능

- [x] 자동차 이동

### 5. 게임 진행 기능
- [x] 0~9 사이 랜덤 숫자 생성
- [x] 한 라운드 게임 진행
- [x] 라운드 횟수만큼 게임 진행
- [x] 게임에 참여한 자동차들의 현재 위치 리스트로 반환(추가)
- [x] 현재까지 가장 많이 이동한 자동차 반환
