# 기능 목록
## 사용자 입력
- [ x ] 사용자는 자동차 이름을 입력할 수 있다.
- [ x ] 사용자는 시도할 횟수를 입력할 수 있다.

## 문구 출력
- [ x ] 실행 결과를 출력할 수 있다.
- [ x ] 최종 우승자를 출력할 수 있다.

## 게임 진행 기능
- [ ] 무작위 0-9 사이의 값을 생성한다.
- [ ] 생성된 값이 4 이상일 경우 전진한다.
- [ ] 우승자를 가려낼 수 있다.

## 검증
### 사용자가 잘못된 입력을 할 경우 IllegalArgumentException을 발생시킨다.
- [ ] 자동자 이름 입력 시 5자 이하만 입력해야 한다.
- [ ] 게임 횟수는 1 이상의 정수만 입력해야 한다.
