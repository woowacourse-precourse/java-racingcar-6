# 기능 구현 목록

## 게임 시작

- [x] 자동차 이름 입력
    - [x] 자동차 이름 입력 문구 출력
    - [x] 사용자 입력 받기
    - [x] 입력이 없을때에 대한 예외 처리
    - [x] 이름이 5자 초과를 초과 할때에 대한 예외 처리
    - [x] 쉼표 사이에 입력이 없는 경우에 대한 예외 처리


- [x] 시도 횟수 입력
    - [x] 시도 횟수 입력 문구 출력
    - [x] 사용자 입력 받기
    - [x] 입력이 없을때에 대한 예외 처리
    - [x] 입력이 숫자가 아닌 경우에 대한 예외 처리

- [x] 입력 데이터 저장
