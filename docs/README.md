# 기능 목록

### 입력
- [X] 사용자는 자동차의 이름을 여려개 입력할 수 있으며 각 이름은 쉼표[,] 로 구분된다
- [X] 사용자는 자동차들이 몇 번의 이동 시도를 할 지 입력할 수 있다.

### 예외 (IlleaglArgumentExecption 발생)

- 자동차 이름 입력
  - [X] 자동차 이름의 길이가 5자를 초과할 경우
  - [X] 자동차의 이름이 서로 중복되는 경우
  - [X] 자동차 이름 입력시 쉼표[,]를 연달아서 입력하는 경우


- 자동차 이동 시도 입력
  - [X] 시도횟수가 문자열일 경우
  - [X] 시도횟수가 1~10000사이 자연수를 만족하지 않을 경우

### 도메인

- 자동차
  - [X] 자동차는 시도별로 0에서 9사이의 무작위 값 중 무작위 값이 4이상일 경우 전진한다.

### 출력

- [X] 각 시도별로 자동차들의 위치 상황을 출력한다.
- [X] 모든 시도를 마친 후에 누가 우승했는지 출력한다. 이 때 우승자는 한 명 이상일 수 있다.
  - [X] 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.
