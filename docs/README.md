# 객체 당 기능 목록

## 입력
- [x] 자동차 이름을 쉼표 기준으로 입력 받기
  - 잘못된 값 입력 받았는 지 검증
    - [x] 자동차 이름으로 공백이 들어오지는 않았는가
- [x] 시도할 회수 입력 받기
  - 잘못된 값 입력 받았는 지 검증
    - [x] 숫자 외에 다른 값이 들어왔는가?
    - [x] 0 이하의 값이 들어왔는가?

## 출력
- [x] 각 차수별 자동차 이름과 이동 결과 출력
- [x] 우승자 출력
  - 단독 우승자
  - 공동 우승자

## 자동차
- [x] 차 이름 받아서 새로운 차 만들어 주기
  - [x] 5글자 이하인지 검증
- [x] 전진하기

## 자동차 리스트
- [x] 리스트에 차 추가하기
  - [x] 중복된 이름의 차 있는지 검증
- [x] 차들 전진시키기
- [x] 우승자 리스트 얻기
- [x] 레이스 결과 얻기

## 경기 진행자
- [x] 레이스 진행하기
